========== BÁO CÁO LỖI VÀ HƯỚNG DẪN CẬP NHẬT MODELS & DTOs ==========
Cơ sở dữ liệu (db.sql) và các Model/DTO hiện tại đang có sự lệch pha lớn. 
Bạn hãy tiến hành sửa các lỗi chi tiết dưới đây trong backend của mình:

1. BẢNG USERS (Lệch cột is_active)
- Trong `backend/Models/User.cs`:
  [Thêm vào]
  [Column("is_active")]
  public bool IsActive { get; set; } = true;

- Trong `backend/DTOs/UserDto.cs` và `backend/DTOs/CreateUserDto.cs`:
  [Thêm vào]
  public bool IsActive { get; set; } = true;

- Trong `backend/DTOs/UpdateUserDto.cs`:
  [Thêm vào]
  public bool? IsActive { get; set; }

2. BẢNG BRANDS (Lệch cột is_active và updated_at)
- Trong `backend/Models/Brand.cs`:
  [Thêm vào]
  [Column("is_active")]
  public bool IsActive { get; set; } = true;
  
  [Column("updated_at")]
  public DateTime UpdatedAt { get; set; } = DateTime.UtcNow;

- Trong `backend/DTOs/BrandDto.cs` và `backend/DTOs/CreateBrandDto.cs`:
  [Thêm vào]
  public bool IsActive { get; set; } = true;
  public DateTime UpdatedAt { get; set; }

- Trong `backend/DTOs/UpdateBrandDto.cs`:
  [Thêm vào]
  public bool? IsActive { get; set; }

3. BẢNG PRODUCTS (Nghiêm trọng: Sai kiểu dữ liệu Status và Thiếu khóa ngoại BrandId)
- Trong `backend/Models/Product.cs`:
  [SỬA] Đổi Status từ kiểu string sang int (vì db.sql là INT)
  [Column("status")]
  public int Status { get; set; } = 1; // 1: draft, 2: published, 3: deleted
  
  [THÊM] Khóa ngoại BrandId (vì db.sql bắt buộc có brand_id UUID NOT NULL)
  [Required]
  [Column("brand_id")]
  public Guid BrandId { get; set; }
  
  [THÊM] Navigation Property (Map với bảng Brand)
  [ForeignKey("BrandId")]
  public Brand Brand { get; set; } = null!;

- Trong `backend/DTOs/ProductDto.cs`:
  [SỬA]
  public int Status { get; set; } = 1;
  [THÊM]
  public Guid BrandId { get; set; }

- Trong `backend/DTOs/CreateProductDto.cs`:
  [SỬA] Chuyển đổi trạng thái từ string sang int
  public int? Status { get; set; }
  [THÊM] Khóa ngoại brand bắt buộc
  public Guid BrandId { get; set; }

- Trong `backend/DTOs/UpdateProductDto.cs`:
  [SỬA] Chuyển đổi trạng thái từ string sang int (nullable cho update)
  public int? Status { get; set; }
  [THÊM] Khóa ngoại brand (nullable)
  public Guid? BrandId { get; set; }
